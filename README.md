# E-commerce Admin API

## Overview
A comprehensive RESTful API for managing e-commerce operations including products, sales, and inventory for Amazon & Walmart platforms.

## Technology Stack
- **Programming Language**: Python 3.12.2
- **Framework**: FastAPI
- **API Type**: RESTful API
- **Database**: Postgres
- **ORM**: SQLAlchemy

## Setup Instructions

### Prerequisites
- Python 3.8 or higher
- pip (Python package manager)

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Vara-Ali/E-commerce-FastAPI.git
   cd E-commerce-FastAPI



2. **Create virtual environment**
```bash
python -m venv venv
```

3. **Activate virtual environment**

Windows:
```bash
venv\Scripts\activate
```
macOS/Linux:
```bash
source venv/bin/activate
```


4. **Install dependencies**
```bash
pip install -r requirements.txt
```


6. **Set up database**

```bash
python main.py
```


8. **Run the application**

```bash
uvicorn main:app --reload
```

10. **Access the API**

API Base URL:
```bash
http://localhost:8000
```
